# Excel工序对比工具

## 功能说明
这是一个用于对比两个Excel表格中工序差异的工具。工具可以：
1. 上传两个Excel文件（基准表和比较表）
2. 自动对比两个表格中的工序差异
3. 显示差异结果，包括：
   - 比较表中新增的工序
   - 比较表中缺少的工序
   - 两表中相同的工序

## 使用方法
1. 运行程序后，点击"选择基准表"按钮上传第一个Excel文件
2. 点击"选择比较表"按钮上传第二个Excel文件
3. 点击"开始对比"按钮查看结果
4. 结果将在界面上显示，并可以导出为新的Excel文件

## 注意事项
- 两个Excel表格必须具有相同的表头结构
- 文件格式支持.xlsx和.xls
